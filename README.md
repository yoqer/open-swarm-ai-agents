## ğŸš€ Getting Started with Open Swarm AI Agents

To begin your journey with Open Swarm AI Agents, ensure you have Python 3.7 or a newer version installed.

**Installation Guide**

```bash
pip install open-swarm-ai-agents
```

**Quick Example**

```python
from open_swarm_ai_agents import Agent, Task

# Define your intelligent agents
research_agent = Agent(name="Researcher", role="Gathers information on AI trends")
writer_agent = Agent(name="Writer", role="Composes engaging content based on research")

# Create a collaborative task
blog_post_task = Task(
    description="Generate a comprehensive blog post about the future of AI, leveraging research findings.",
    agents=[research_agent, writer_agent],
)

# Execute the multi-agent task
blog_post_task.run()
```
